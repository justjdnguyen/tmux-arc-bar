# tmux-arc-bar

A minimal tmux status bar plugin with a blue-to-white gradient arc on both sides.

## Install via TPM

Add to your `~/.tmux.conf`:

```bash
set -g @plugin 'justjdnguyen/tmux-arc-bar'
```

Then press `prefix + I` to install.

## Manual Install

Clone and source:

```bash
git clone https://github.com/justjdnguyen/tmux-arc-bar.git ~/.tmux/plugins/tmux-arc-bar
tmux source ~/.tmux.conf
```
